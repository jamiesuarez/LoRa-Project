Welcome to the LoRa Web Server!

LoRaWAN is a telecommunication protocol that enables low-speed radio communication of low-power objects communicating with LoRa technology connected to the Internet via gateways, thereby contributing to the Internet of Things. This mini-gateway enables you to transmit messages to any of the 8 receivers through 1 transmitter. The set is controlled by a RaspberryPI 3 card which loads a rudimentary WEB server in order to have access, via WiFi, to the content of the received messages and to be able to trigger the transmission of a predefined message.
